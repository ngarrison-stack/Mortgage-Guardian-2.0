## Asset Guidelines

- **SVG icons**: Use consistent stroke width and style
- **Images**: Optimize for web (WebP format when possible)
- **Logos**: Provide both light and dark versions
- **Mobile**: Ensure all images look good on mobile devices
- **Accessibility**: Include alt text for all images

## Image Optimization

Before deployment, optimize all images:
- Compress PNG/JPG files
- Convert to WebP when possible
- Use appropriate dimensions (don't serve oversized images)
- Include multiple sizes for responsive images
