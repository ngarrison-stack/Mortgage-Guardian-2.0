<feature>
  <name>Confidence Scoring & Evidence Linking</name>
  <files>backend-express/services/confidenceScoringService.js, backend-express/__tests__/services/confidenceScoringService.test.js</files>
  <behavior>
    calculateConfidence(aggregatedData) → { overall, breakdown: { documentAnalysis, forensicAnalysis, complianceAnalysis } }

    Cases:
    - Clean case (no issues) → overall: 95-100, all breakdowns high
    - High anomalies + discrepancies → overall: 20-40 (degraded)
    - Mixed (some anomalies, no compliance violations) → proportional scoring
    - Missing forensic data → forensicAnalysis: null, overall calculated from available layers (weights redistributed)
    - Missing compliance data → complianceAnalysis: null, weights redistributed
    - Empty case → overall: 100 (no evidence of problems = high confidence in clean status)

    documentAnalysisScore(documentAnalyses[]) → 0-100
    Cases:
    - All docs complete, no anomalies → ~100
    - Low completeness, many anomalies → low score
    - Empty array → 100 (no docs analyzed = no negative evidence)

    forensicAnalysisScore(forensicReport) → 0-100
    Cases:
    - No discrepancies → ~100
    - Critical discrepancies → heavily penalized
    - null input → null (not 0 — indicates missing, not bad)

    complianceAnalysisScore(complianceReport) → 0-100
    Cases:
    - No violations → ~100
    - Critical federal violations → heavily penalized (severity multiplier)
    - State violations compound federal → lower score

    determineRiskLevel(overallScore) → 'critical' | 'high' | 'medium' | 'low' | 'clean'
    Cases:
    - Score 0-25 → critical
    - Score 26-50 → high
    - Score 51-70 → medium
    - Score 71-90 → low
    - Score 91-100 → clean

    buildEvidenceLinks(aggregatedData) → evidenceLinks[]
    Cases:
    - Anomaly → links to source documentId
    - Discrepancy → links to both document IDs in the pair
    - Violation → links to statute + source finding + source documents
    - Payment issue → links to Plaid transaction + document
    - No findings → empty array
  </behavior>
  <implementation>
    Singleton service using config from consolidatedReportConfig.js.
    - calculateConfidence: weighted sum of per-layer scores with redistribution for missing layers
    - Per-layer scorers: start at 100, subtract penalties based on finding count and severity
    - buildEvidenceLinks: iterate all finding types, create evidence link objects with findingId, type, sourceDocumentIds, description
    - determineRiskLevel: simple threshold lookup from RISK_THRESHOLDS config
    Mock nothing — this is pure calculation logic. Test with constructed input data.
  </implementation>
</feature>
