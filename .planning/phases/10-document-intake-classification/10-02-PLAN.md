**1. Install dependency:**
```bash
cd backend-express && npm install pdf-parse
```

pdf-parse is a lightweight, pure-JS PDF text extractor — no native binaries, serverless-compatible. Use it for text-based PDFs. For scanned PDFs and images, use Claude Vision (already available via @anthropic-ai/sdk).

**2. Create ocrService.js:**

```
const pdfParse = require('pdf-parse');
const Anthropic = require('@anthropic-ai/sdk');
const { createLogger } = require('../utils/logger');
const logger = createLogger('ocr');
```

**OcrService class with methods:**

- `extractText(fileBuffer, fileName)` — Main entry point. Detects file type and routes to appropriate extractor:
  - PDF → `_extractFromPdf(fileBuffer)`
  - Image (jpg, jpeg, png, heic, tiff) → `_extractFromImage(fileBuffer, mimeType)`
  - Throw error for unsupported types
  - Returns: `{ text, method, pageCount, confidence }` where method is 'pdf-parse' or 'claude-vision'

- `_extractFromPdf(fileBuffer)` — Try pdf-parse first. If extracted text is meaningful (>50 chars after trimming whitespace), return it as method='pdf-parse'. If text is empty/negligible (scanned PDF), fall back to `_extractFromImage(fileBuffer, 'application/pdf')` with method='claude-vision'.

- `_extractFromImage(fileBuffer, mimeType)` — Send image to Claude Vision API:
  ```
  client.messages.create({
    model: 'claude-sonnet-4-5-20250514',
    max_tokens: 8192,
    messages: [{
      role: 'user',
      content: [
        {
          type: 'image',
          source: {
            type: 'base64',
            media_type: mimeType,
            data: fileBuffer.toString('base64')
          }
        },
        {
          type: 'text',
          text: 'Extract ALL text from this document image. Preserve the original formatting, layout, and structure as closely as possible. Include all numbers, dates, names, addresses, and financial figures exactly as they appear. Return only the extracted text, no commentary.'
        }
      ]
    }]
  })
  ```
  Return extracted text with method='claude-vision'.

- `_detectFileType(fileName)` — Parse extension to determine if PDF or image. Use extension-based detection (matching the pattern in documentService.getContentType). Return { isPdf, isImage, mimeType }.

**Important implementation notes:**
- Initialize Anthropic client same as claudeService.js (from ANTHROPIC_API_KEY env var)
- If ANTHROPIC_API_KEY is not set, the Vision extraction should throw a clear error ("OCR requires ANTHROPIC_API_KEY for image processing")
- For PDF extraction via pdf-parse: wrap in try/catch, log errors, throw with context
- The 50-char threshold for "meaningful text" avoids treating scanned PDFs with embedded metadata-only text as text PDFs
- Use model 'claude-sonnet-4-5-20250514' for Vision (cost-effective for OCR, fast)
- Log extraction method, text length, and processing time for each extraction

**Singleton export:** `module.exports = new OcrService();`
