   //
   // Calibration anchors:
   //   - 1 critical anomaly: score drops ~30 points (CLEAN→MEDIUM or LOW→HIGH)
   //   - 3 medium anomalies: score drops ~24 points (similar to 1 high)
   //   - 1 critical compliance violation: score drops ~35 points

2. Update LAYER_SCORING_FACTORS (anomaly penalties):
   - critical: 30 (was 40 — too aggressive, single anomaly shouldn't dominate)
   - high: 22 (was 30)

   - low: 5 (was 10 — low anomalies should barely register)
   - info: 0 (unchanged)
   - Rationale: Old values meant 3 medium anomalies (60 points) exceeded 1 critical (40 points), which is backwards.

3. Update forensic discrepancy penalty:
   - Keep at 20 per severity unit (reasonable — each discrepancy IS significant)

5. Update the floor drag cap from 45 to 35:
   - Rationale: If any scoring component hits zero, the overall situation IS dire. Capping at 45 was too generous — a document where forensic analysis found zero confidence should score much lower.
